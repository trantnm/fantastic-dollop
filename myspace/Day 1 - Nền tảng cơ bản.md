- ++a : tăng giá trị của a lên 1 đơn vị trước khi thực hiện tính toán

```
a = 5 -> a = 5
b = 2 + (++a) -> a = 6, b = 8
```

- a++: tăng giá trị của a lên 1 đơn vị sau khi đã tính toán xong

```
a = 5
b = 2+(a++) -> b = 7, a = 6
```
